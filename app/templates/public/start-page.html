GitHub & Trello Powerup

This powerup creates an integration between GitHub and Trello that helps to synchronise the two tools for agile, digital teams.

These features are available through the powerup:

  • require product signoff before code is merged
  • list pull requests on Trello tickets

Setting up the powerup takes around 5 minutes.

Before you start

You will need to be signed in to the following services:

  • GitHub, with admin permissions to your team's repositories
  • Trello, with admin permissions on your team's boards
